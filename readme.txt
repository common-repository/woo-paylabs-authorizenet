== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/woo-paylabs-authorizenet' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Authorize Net WooCommerce payment gateway => Settings screen to configure the plugin
